# Inventario de Farmacia

Aplicación de escritorio desarrollada en Python con CustomTkinter para gestionar medicamentos en un inventario. Permite agregar, buscar, eliminar y listar medicamentos, además de guardar los datos en un archivo JSON.

## Funcionalidades

- Agregar medicamentos con:
  - Nombre
  - Cantidad
  - Uso
  - Fecha de vencimiento
- Buscar medicamentos por nombre.
- Eliminar medicamentos por nombre.
- Listar todos los medicamentos en el inventario.
- Guardar y cargar inventario automáticamente en un archivo `inventario.json`.

## Tecnologías

- Python 3.x
- CustomTkinter
- JSON para almacenamiento local
